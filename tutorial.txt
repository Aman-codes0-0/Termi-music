=======================================
   TUI Music Player - Quick Tutorial
=======================================

1. PREREQUISITES
----------------
Make sure you have Python 3.10 or newer installed on your computer.

2. INSTALLATION
---------------
Open your terminal and navigate to this project folder, then run:

# Create a virtual environment
python3 -m venv venv

# Activate the virtual environment
# On Linux/macOS:
source venv/bin/activate
# On Windows:
venv\Scripts\activate

# Install the required libraries
pip install -r requirements.txt

3. NAVIGATING MUSIC
-------------------
Use the built-in Directory Explorer on the left panel to browse your computer! 
Just click or press Enter on any folder containing `.mp3`, `.wav`, `.flac`, or `.ogg` files to instantly load them into your playlist.

4. RUNNING THE PLAYER
---------------------
Once the virtual environment is activated, simply run:
python main.py

5. CONTROLS
-----------
When the Terminal UI opens, you can use your mouse or your keyboard:
- Use the 'Tab' and 'Shift+Tab' keys to navigate between panels and buttons.
- Select a folder from the tree on the left to automatically scan it.
- Select a song from the table on the right to play it.
- Press the 'Spacebar' keyboard shortcut at any time to toggle Pause/Resume.
- Press 'd' to open the Themes selector (your choice will be saved permanently!).
- Use Play, Next, and Previous buttons or mouse clicks for active navigation.

6. BUILDING AN EXECUTABLE
-------------------------
To build a standalone executable file for your current OS:
pyinstaller main.spec
The binary will be generated inside the `dist/` folder.
